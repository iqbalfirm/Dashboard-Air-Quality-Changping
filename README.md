# Dashboard-Air-Quality-Changping
Dashboard interaktif kualitas udara di distrik Changping dari 2013 - 2017

## 1. Pastikan sudah mempunyai semua file yang dibutuhkan
``` Dokumen yang dibutuhkan
dashboard
|__data_polusi.csv
|__dashboard.py
|__requirements.txt
```
## 2. Buka lokasi folder
```
cd lokasi/folder/dashboard
```
## 3. Buat virtual environntmen dan aktifkan
```
python -m venv venv
```
**Mengaktifkan virtual environtmen pada windowns**
```
venv\Scripts\activate
```
**Mengaktifkan virtual environtmen pada mac dan linux**
```
source venv/bin/activate
```
## 4. Install library yang diperlukan
```
pip install -r requierements
```
## 5. Jalankan Dashboard
```
streamlit run dashboard.py
```
